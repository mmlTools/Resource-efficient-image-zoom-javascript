Data elements:

  • src - Full or relative path to the thumbnail
  • data-toggle="image-zoom" - Tells the script that this image can be zoomed
  • data-src - Full or relative path to the big image
  • data-caption - Title of the image, if not set, it won't show anything
  • data-description - Description of the image, if not set, it won't show anything
